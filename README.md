# morning 
面向用户
> 有一定动手能力的普通用户和程序员  

使用场景
> 是打开terminal时自动对某些网站监测并进行规律化操作(如 签到，打卡，发早安)

支持平台
> 主要是 MacOS和Linux

## 登录逻辑
登陆部分选择省去，由用户将浏览器的cookies提取出来。目前的可选项是:

* [x] Chrome + EditThisCookie
* [ ] 自动从 Chrome 浏览器内获取Cookies (系统默认路径下)
* [x] 自动从 Firefox 浏览器内获取Cookies (系统默认路径下)
    * [ ] 网易音乐无法登陆
